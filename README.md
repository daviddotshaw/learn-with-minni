# Learn with Minni

A nostalgic kids learning game, originally created in the early 1990s, rebuilt as a modern Progressive Web App (PWA).

## About

Children tap different parts of Minni Mouse's body to hear the name of each body part spoken aloud. It's simple, colourful, and designed to be fun for young children.

**Body parts:** Ears, Nose, Tummy, Hands, Feet

## How to play

1. Open the app — Minni introduces herself
2. Tap **Touch here to start!**
3. Tap any part of Minni's body to hear its name
4. Tap **Quit** to return to the start screen

## Play it

👉 **[daviddotshaw.github.io/learn-with-minni](https://daviddotshaw.github.io/learn-with-minni/)**

Works on desktop, tablet, and mobile. Can be installed as an app via your browser's "Add to Home Screen" option.

## Features

- Original audio assets from the 1990s game
- Installable as a PWA (works offline once loaded)
- Touch and mouse support
- Responsive design — works on any screen size

## Original game

The original *Learn with Minni* was a DOS-based children's educational game from the early 1990s. This PWA is a faithful recreation using the original image and sound assets.
